# Setup
- Have a recent g++/clang setup
- Have git setup
 ```
git clone --recurse-submodules https://github.com/Mk-Chan/gw2combat
./setup.sh
./compile.sh
  ```
# Run the program
```
./build/debug/gw2combat
```

- Currently only runs basic raid core-guard greatsword-only aa-chain only benchmark
